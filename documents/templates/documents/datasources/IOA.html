IOA Update

    Source
  • Code Page: 65001 (UTF-8)
  • Format: Delimited
  • Text Qualifier: "
  • Head Row Delim: {CR}{LF}
  • Row Delim: {LF}
  • Column Delim: ,
  • Column names are in the first data row
  • The structure of the International Odontoglossum Alliance registration files will be as follows.
    • RegistrationID int NULL,
    • SeedGenus nvarchar(50) NOT NULL,
    • SeedGenusName nvarchar(50) NOT NULL,
    • SeedGrexEpithet nvarchar(255) NOT NULL,
    • PollenGenus nvarchar(50) NOT NULL,
    • PollenGenusName nvarchar(50) NOT NULL,
    • PollenGrexEpithet nvarchar(255) NOT NULL,
    • GrexGenus nvarchar(50) NOT NULL,
    • GrexGenusName nvarchar(100) NOT NULL,
    • GrexEpithet nvarchar(255) NOT NULL,
    • ApplicantTitle nvarchar(10) NULL,
    • ApplicantFirstName nvarchar(50) NOT NULL,
    • ApplicantSecondName nvarchar(50) NOT NULL,
    • ApplicantCompanyName nvarchar(255) NULL,
    • OriginatorFirstName nvarchar(50) NOT NULL,
    • OriginatorLastName nvarchar(50) NOT NULL,
    • DateRegistered date NULL,
    • RegistrationSource varchar(3) NOT NULL
  • The IOA will only publish with a RegistrationSource of 'IOA', this seperates the IOA data from other registration sources (e.g. RHS).
  • The IOA will publish data with the file name convention of Wikiregistration_yyyyMMdd.csv to indicate the year month and day the file was published.
