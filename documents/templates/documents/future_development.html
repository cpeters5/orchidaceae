React-Redux

React is a single page application (typically index.html), which could be a complex object with multiple components, each dynamically changing in response to user's request. React handles asynchronous and mutations.

Redux is a predictable container for javascript applications. Redux is a middletier between React (frontend) and API. React generates asynchronous and mutations requests. Redux handles the state of data. It puts restrictions on how and when mutations (to each component). It sends requests to backend in a consistent fashion and receive data responses from API which is then handed back to React.
